### How I build it and How can You Build this

## Requirement:

* ALuminium Extrusions of-
  * 2020 size- 380mm x2
  * 400mm x2
  * 2040 size- 150mm x2
  * 420mm x1
* Corner Brackets x4
* Inner Joints x6
* Sliding Nuts [as per your need, read the instructions first for this]
* Hot Glue
* Bolts
* Nuts
* 2020 Gantry Plates x4
* Derlin V wheels x16
* Nema 17 Stepper Motor x4
* Nema 17 Mounting bracket (L shaped) x4
* GT2 20 teeth Pulley x2
* 10mm timing belt (open)
* 10mm Idler Pulley x5
* 5mm to 8mm Coupler x1
* Lead Screw Tr8x2 x1
* Arduino Mega x1
* Ramps 1.4 x1
* Stepper Driver x4
* Power Supply (12v)
* Endstop swichtes
* Zip Ties
* MK8 hot end
* MK8 extruder

## Now for the electronics

Stack your ramps onto the arduino mega..
Then wire everything according to this diagram--

<img width="542" height="218" alt="image" src="https://github.com/user-attachments/assets/aedf122b-9250-4fa9-bb6a-44f7b87a84e4" />

For this build though, I will not be using LCD and electronics fan..
Connect the stepper motor drivers onto the RAMPS and power the RAMPS with the 12V SMPS..

NOW--
Open the Marlin Firmware folder and inside find a Marlin folder and open it-
look for a Marlin.ino file in this folder. This is the main file to be uploaded to Arduino Mega..
But before that looks for a Configuaration.h file in that same folder and open it (open in notepad)
now look for
```
#define DEFAULT_AXIS_STEPS_PER_UNIT { 80, 80, 400, 93 }
```
&
```
#define X_BED_SIZE 220
#define Y_BED_SIZE 220
```
these must be adjusted by you for working of the printer..
if your build is just like mine, then you have to put these values-
```
#define DEFAULT_AXIS_STEPS_PER_UNIT { 80, 80, 1600, 93 }
```
&
```
#define X_BED_SIZE 200   
#define Y_BED_SIZE 150
```
and now you are all SET.


Just open the marlin.ino and upload the code to arduino mega...

Then open Printrun folder and open the exe file in it...

from here you can control and give print commands..
